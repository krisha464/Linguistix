# ✨ Linguistix

**Linguistix** is a powerful, minimal, and intelligent language translation and learning application built with Python and Streamlit. It offers real-time translation, AI-powered conversations, a visual dictionary, and a fun gaming zone to help you master new languages.

## 🚀 Features

- **🌐 Professional Translator**: Multi-language support with document and image translation capabilities.
- **🤝 AI Conversation**: Engage in real-time bilingual dialogue powered by advanced AI.
- **📖 Visual Dictionary**: Explore words with synchronized definitions, thesaurus (synonyms/antonyms), and AI-generated visual aids.
- **📲 Offline Hub**: Save phrasebooks for travel and access them without an internet connection.
- **📸 Image & Doc Translate**: Extract and translate text from images (OCR) and documents (PDF, Word).
- **🧩 Fun Zone**: Interactive word games like Word Quest, Pair Finder, and Detective to make learning addictive.
- **🎨 Premium UI**: A stunning, theme-reactive interface with glassmorphism effects and smooth animations.

## 🛠️ Tech Stack

- **Framework**: [Streamlit](https://streamlit.io/)
- **Translation Engine**: `deep-translator`
- **Speech**: `gTTS`
- **OCR**: `easyocr`
- **Document Processing**: `PyMuPDF`, `python-docx`, `fpdf`
- **Thesaurus**: `Datamuse API`
- **Visuals**: `Pollinations AI`, `Icons8`

## 📦 Installation & Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Linguistix.git
   cd Linguistix
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   streamlit run app.py
   ```

## ☁️ Deployment

Linguistix is ready to be deployed on **Streamlit Community Cloud**. 
1. Push your code to GitHub.
2. Sign in to [Streamlit Cloud](https://share.streamlit.io/).
3. Click "New App", select your repo, and set the Main file path to `app.py`.
4. Click **Deploy**!
